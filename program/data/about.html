This tool allows you to spatially plot expressed genes and transcriptional signatures on various datasets generated using spatial transcriptomics technology (Visium)

Publication: XYZ

10X Visium: This technology allows for the capture of whole transcriptome information from tissue sections while preserving specific tissue context.

Tissues: These datasets come from the following models:

  1. Healthy rat colon
  2. Syngeneic mouse tumor models (B16, MC38)
  3. Human PDAC (Donors A, B, and C)

Signatures: These gene signatures were gathered from tissue- and indication- specific publications and curated manually. (See Table X for more information about the signatures.)

Feedback: We welcome any feedback including signature and data issues -- please contact Anna Lyubetskaya and Eugene Drokhlyansky.

This Shiny application utilizes CanvasXpress to provide interactive performant visualizations. For more information on this technology visit https://canvasxpress.org/
